Supported API Listing
Core Registry API (Version 3)
The primary API for accessing Apicurio Registry. This is the most recent API and is preferred for all use-cases. All other APIs are supported for compatibility purposes only.
/apis/registry/v3
Core Registry API (Version 2)
The primary API for accessing Apicurio Registry. This is the previous major version release of the core API and should now be considered deprecated. Please upgrade to using version 3 of the core API as soon as possible.
/apis/registry/v2
Confluent Schema Registry API (Version 7)
To provide compatibility with Confluent Serdes (and other clients), Apicurio Registry implements the API defined by the Confluent Schema Registry. Use this endpoint if your tooling is compatible with v7 of that API. This is only provided for compatibility purposes, the core Registry API should be used instead when possible.
/apis/ccompat/v7
CNCF Schema Registry API (Version 0)
The CNCF community has worked to create an open standard for schema registries, for cloud events and for other use-cases. Apicurio Registry implements the API defined by this work.
/apis/cncf/v0
