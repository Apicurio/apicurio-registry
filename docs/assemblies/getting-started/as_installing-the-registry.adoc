= Installing Apicurio Registry

Apicurio Registry is a datastore for event schemas and API designs. This topic explains how to install and run Apicurio Registry container images for each of the following storage options:

* In-memory
* Java Persistence API
* Apache Kafka
* Apache Kafka Streams
 
.Prerequisites
* * xref:intro-to-registry[]
* You must have Docker installed and the Docker daemon running.

.Additional resources
* For details on building from source, see https://github.com/Apicurio/apicurio-registry
