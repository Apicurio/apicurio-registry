# Apicurio Registry `.project` Directory

This directory contains the CNCF `.project` metadata for the Apicurio Registry project.

For documentation on the directory structure, schema, and tooling, see the
[CNCF Automation repository](https://github.com/cncf/automation/tree/main/utilities/dot-project).

When the `.project` GitHub repository is created for Apicurio Registry under the CNCF
GitHub Enterprise account, the contents of this directory should be moved there.
